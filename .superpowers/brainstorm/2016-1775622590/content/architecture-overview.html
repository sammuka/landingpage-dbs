// Por que a solução cloud foi escolhida
A Assurant avaliou duas rotas: migração conservadora on-premises (Java 25 + Spring Boot + Oracle intacto + WildFly) e reescrita cloud-native (.NET 9 + Azure SQL + Azure Container Apps). A migração on-premises preservaria o Oracle e reduziria o risco de dados, mas manteria a complexidade operacional de um servidor de aplicação. A rota cloud elimina o host-parity hack dos jobs Quartz, entrega escalabilidade automática via KEDA, externaliza segredos no Key Vault e integra Okta nativamente — eliminando todas as vulnerabilidades críticas por design.
